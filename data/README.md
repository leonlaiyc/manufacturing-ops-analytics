# Data Provenance & Honest Scope

## Layer 1 — Real (raw/, gitignored)

**4TU "Production Analysis with Process Mining Technology"** manufacturing
production event log. A real production process exported from an ERP system;
activities correspond to stations/steps on a manufacturing line.

- Source: 4TU.ResearchData repository.
- Not committed to the repo (license/redistribution); download locally into `raw/`.
- Used in M1 for process discovery, cycle-time, bottleneck and rework analysis.

## Layer 2 — Synthetic (synthetic/)

Generated by `src/generator`. A transparent, hand-written discrete-event
production-line model whose structure is informed by the publicly available
**SMT2020 semiconductor manufacturing testbed**.

**This data is synthetic and clearly labeled.** It is not real fab data and is
not intended to predict any real fab. Its purpose is to demonstrate methodology
with a known ground-truth bottleneck and injectable, labeled anomalies, so that
detection and monitoring methods can be objectively validated.

All generation is seeded for reproducibility.
